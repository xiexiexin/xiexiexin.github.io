[原]运维基础——Zabbix入门

zabbix是一个基于web界面的、提供分布式系统监视及网络监视功能的企业级开源解决方案

zabbix 由2部分组成：

  • zabbix server

  • zabbix agent

zabbix server可以通过SNMP、zabbix agent、ping、端口监视等方法提供对远程服务器/网络状态的监视，数据收集等功能，它可以运行在多个操作系统上。

试验环境

centos 7

安装（编译安装方式）

LAMP环境
